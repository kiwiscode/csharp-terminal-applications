## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/kiwiscode/csharp-terminal-applications.git
   cd csharp-terminal-applications
   ```

2. Navigate to the desired project:

   ```js
   cd "project-folder-name"
   ```

3. Build and run:
   ```bash
   dotnet run
   ```
